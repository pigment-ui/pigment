## Initial requirements

- Create your Next.js application with Tailwind CSS.
- Install needed packages which will be used for creating components.

```bash
npm i react-aria react-aria-components tailwind-variant tailwind-merge @radix-ui/react-slot @radix-ui/react-icons
```

## Colors configuration

<Steps>

### Color variables

Create color variables in CSS for both light and dark mode. See more on [colors](/docs/overview/colors) page.

`color.css`

```css
:root {
  --default-0: 255 255 255;
  --default-50: 250 250 250;
  // ...

  --info-50: 239 246 255;
  --info-100: 219 234 254;
  // ...

  --success-50: 240 253 244;
  --success-100: 220 252 231;
  // ...

  --warning-50: 254 252 232;
  --warning-100: 254 249 195;
  // ...

  --error-50: 254 242 242;
  --error-100: 254 226 226;
  // ...
}

:root[class="dark"] {
  --default-0: 0 0 0;
  --default-50: 10 10 10;
  // ...

  --info-50: 23 37 84;
  --info-100: 30 58 138;
  // ...

  --success-50: 5 46 22;
  --success-100: 20 83 45;
  // ...

  --warning-50: 66 32 6;
  --warning-100: 113 63 18;
  // ...

  --error-50: 69 10 10;
  --error-100: 127 29 29;
  // ...
}
```

### Global CSS

Import `color.css` to your global CSS file.

`globals.css`

```css
@import "./color.css";
@tailwind base;
@tailwind components;
@tailwind utilities;

// ...
```

### Tailwind config

`tailwind.config.ts`

```ts
theme: {
  // ...
  extend: {
    // ...
    color: {
      default: {
        0: "rgb(var(--default-0) / <alpha-value>)",
        50: "rgb(var(--default-50) / <alpha-value>)",
        // ...
      },
      info: {
        50: "rgb(var(--info-50) / <alpha-value>)",
        100: "rgb(var(--info-100) / <alpha-value>)",
        // ...
      },
      success: {
        50: "rgb(var(--success-50) / <alpha-value>)",
        100: "rgb(var(--success-100) / <alpha-value>)",
        // ...
      },
      warning: {
        50: "rgb(var(--warning-50) / <alpha-value>)",
        100: "rgb(var(--warning-100) / <alpha-value>)",
        // ...
      },
      error: {
        50: "rgb(var(--error-50) / <alpha-value>)",
        100: "rgb(var(--error-100) / <alpha-value>)",
        // ...
      },
    },
  },
}
```

</Steps>

## Project configuration

### Bundle

You can [download](https://download-directory.github.io/?url=https://github.com/pigment-ui/pui-web/tree/main/src/ui) all components as a bundle.

### Manual

- Add [`types.ts`](https://github.com/pigment-ui/pui-web/blob/main/src/ui/types.ts), [`styles.ts`](https://github.com/pigment-ui/pui-web/blob/main/src/ui/styles.ts) and [`utils.ts`](https://github.com/pigment-ui/pui-web/blob/main/src/ui/utils.ts) files to your project in **the same directory** with your components.
- Add other components to your project by making copy and paste manually.
