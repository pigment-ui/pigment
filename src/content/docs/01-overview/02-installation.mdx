## Default installation

<Steps>

### Package installation

- [Create](https://tailwindcss.com/docs/guides/nextjs) your Next.js application with Tailwind CSS.
- Install Pigment UI package.

```bash
npm install pigment-ui
```

### Color variables

Import color variables in the root layout. See more on [colors](/docs/overview/colors) page.

`layout.tsx`

```tsx
// ...
import "pigment-ui/colors";

// ...
```

### Tailwind configuration

Update your Tailwind configuration file.

`tailwind.config.ts`

```ts
import { twConfig } from "pigment-ui";
import type { Config } from "tailwindcss";

const config: Config = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx,mdx}",
    // add path to Pigment UI components
    "./node_modules/pigment-ui/**/*.{js,ts,jsx,tsx,mdx}",
  ],
  // add tailwind config of Pigment UI as preset
  presets: [twConfig],
  darkMode: "class",
  theme: {
    extend: {
      // ...
    },
  },
};

export default config;
```

</Steps>
