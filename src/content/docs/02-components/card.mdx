### Style slots

```ts
type CardStyleSlots = "base" | "header" | "body" | "footer" | "heading" | "buttons";
type CardStyleProps = {
  classNames?: { [slot in keyof CardStyleSlots]?: string };
  styles?: { [slot in keyof CardStyleSlots]?: CSSProperties };
};
```
